# Test Technique

## Bonnes pratiques utilisées

 -  Architecture **MVVM-C**
 - Concurrency: Utilisation ***async/await***
 - **Injection** de **dépendance** par **initialisation**
 - **Protocol-Oriented programming**
 - Test Unitaires (***XCTestCase***)
 - Programmation réactive
